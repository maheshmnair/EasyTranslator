## Features
- Translate text between languages
- Simple, clean UI (HTML/CSS/JS)
- Optional backend proxy (see `backend.js`) to keep API keys server-side


## Quick start
1) **Clone**
```bash
git clone https://github.com/maheshmnair/EasyTranslator.git
cd EasyTranslator
```

2) **Run**

- **Frontend only:**  Open `index.html` in your browser.
- **With backend proxy (recommended for API keys):**
  - Set your Gemini API key in the server code or as an environment variable.
  - Start the server and open the URL it prints.
